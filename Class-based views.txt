Class-based views. Классы и их значения


class View:

    Основной класс в CBV от которого и происходят все остальные классы


class ListView:

    Класс ListView ссылается на отображение нескольких объектов из базы данных. По умолчанию

        ListView отображает все объекты в том порядке, в котором они были созданы. В случае, если требуется изменить

        последовательность этих объектов или же их порядок, то метод get_queryset должен быть переопределен


class CreateView:

    Класс CreateView ссылается на создание формы объекта, его сохранения и повторного отображения

        в случае ошибки в базе данных.


class DetailView:

    Класс DetailView ссылается на отображение одного объекта базы данных. По умолчанию DetailView

        будет отображать только поля объекта. В случае, если требуется изменить эти данные перед отправкой в шаблоны,

        данные объекта могут быть переопределены


class UpdateView:

    Класс UpdateView ссылается на логику обновления конкретного объекта из базы данных с некоторыми

        дополнительмными деталями. Используется для обновления записей в базе данных


class TemplateView:

    Класс TemplateView используется в случае, если нужно представить некоторую информацию на

        HTML-странице. TemplateView не используется в случае, если страница имеет формы и создает или обновляет объекты
